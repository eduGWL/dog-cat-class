# 🐶🐱 강아지 vs 고양이 분류기 (Dog vs Cat Classifier)

이 프로젝트는 Xception 딥러닝 모델(`best_model_xception.keras`)을 사용하여 사용자가 업로드한 이미지가 강아지인지 고양이인지 판별해 주는 간단한 웹 서비스입니다.

## 파일 구성

- `app.py`: Flask 웹 프레임워크 기반의 서버 및 이미지 분류 로직
- `best_model_xception.keras`: 사전 학습된 이미지 분류 모델 (Xception 기반)
- `requirements.txt`: 서비스 실행에 필요한 Python 라이브러리 목록
- `.gitignore`: Git 버전 관리에서 제외할 파일 설정

## 설치 및 실행 방법

아래 안내에 따라 로컬 환경에서 웹 서비스를 구동할 수 있습니다.

### 1. 패키지 설치
먼저 터미널을 열고, 프로젝트 폴더 경로에서 다음 명령어를 입력하여 필요한 라이브러리를 설치합니다.
(가상환경을 사용하시는 경우, 가상환경을 활성화한 후 진행해주세요.)

```bash
pip install -r requirements.txt
```

### 2. 웹 서버 실행
패키지 설치가 정상적으로 완료되면, 다음 명령어로 Flask 서버를 실행합니다.

```bash
python app.py
```

### 3. 접속하기
서버가 정상적으로 구동되고 나면 브라우저를 열고 다음 주소로 접속합니다.
👉 **http://localhost:5000**

접속 후 이미지를 업로드하고 **분류하기** 버튼을 누르시면, AI 모델이 해당 이미지를 분석해 결과를 웹 화면에 출력해 줍니다.
